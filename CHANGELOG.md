# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-11-03

### Added
- Dark theme support with toggle switch
- Multi-language support (Russian and English)
- Improved toast notifications system
- Better synchronization between browser windows
- Enhanced UI with modern design
- Real-time statistics display
- Settings persistence across browser sessions

### Changed
- Complete rewrite of the popup interface
- Improved content script performance
- Better error handling and logging
- Enhanced detection algorithms for AI instructions
- More reliable DOM mutation observation

### Fixed
- Issues with counter synchronization
- Theme persistence problems
- Language switching bugs
- Performance issues on large pages

### Removed
- Legacy code and unused functions
- Test files and development artifacts

## [1.0.0] - 2024-10-15

### Added
- Initial release
- Basic AI instruction removal functionality
- Simple popup interface
- Counter for removed elements
- Chrome extension manifest v3 support

## How to Update

1. Download the latest version from the [Releases](https://github.com/S1ii/coursera-ai-instructions-remover/releases) page
2. Remove the old version from your browser
3. Load the new version in developer mode
4. Your settings and statistics will be preserved automatically

## Support

If you encounter any issues with the update, please:
1. Check the [Issues](https://github.com/S1ii/coursera-ai-instructions-remover/issues) page
2. Create a new issue with detailed information
3. Include your browser version and extension version
